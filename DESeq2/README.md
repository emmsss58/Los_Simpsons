# Análisis global de expresión diferencial

El análisis de expresión diferencial realizado mediante DESeq2 permitió comparar los perfiles transcriptómicos entre los grupos Obeso1 y Obeso2. 
A diferencia de estudios clásicos obesidad versus normopeso, en este caso se compararon dos perfiles obesos distintos, lo que permite explorar posibles 
diferencias metabólicas y genéticas dentro de un mismo fenotipo general de obesidad.

Los resultados obtenidos muestran que existen patrones diferenciales de expresión génica entre ambos grupos, especialmente en genes relacionados 
con regulación del apetito, señalización leptina-melanocortina y metabolismo energético.

Aunque el tamaño muestral fue reducido (2 muestras en Obeso1 y 3 muestras en Obeso2), el análisis permitió identificar tendencias biológicas
coherentes y reproducir correctamente el flujo de trabajo típico de RNA-seq.
